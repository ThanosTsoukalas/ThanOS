Welcome to ThanOS!

ThanOS is an HTML workspace, designed to be as light as possible for your computer. Here, you can use the original versions of very heavy programs like Photshop or DavinCi Resolve, while also maintaining extremely high performance. ThanOS is also great for Web Browsing, conferencing via the web, and light gaming.
